# Anand — Fullstack Developer (Portfolio)

This is a minimal Vite + React portfolio scaffold for Anand.

Requirements
- Node.js (16+)

Install and run locally

```bash
npm install
npm run dev
```

Build

```bash
npm run build
npm run preview
```

Files
- `index.html` — app entry
- `src/` — React source (components and styles)

Edit `src/components` to add your projects, links and real contact details.
